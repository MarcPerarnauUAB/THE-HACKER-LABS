## 🧠 Metodología OSINT
En cada investigación sigo este flujo de trabajo para asegurar que no se pierda ninguna pista:

1.  **Identificación del Objetivo:** Análisis de la pista inicial (Imagen, Usuario, Email o Dominio).
2.  **Recolección de Datos:** Uso de herramientas automáticas y manuales (Dorking, WHOIS, DNS).
3.  **Pivotaje:** Conectar un dato encontrado con una nueva fuente (ej: de un nombre de usuario a una red social).
4.  **Verificación:** Confirmar que la información es verídica y no un "falso positivo".
5.  **Conclusión:** Reporte del hallazgo y resolución del reto.
